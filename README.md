# Mother Language

A website dedicated to the Mother Language Movement.

## Introduction

This project serves as a tribute and informational resource about the Mother Language Movement (Bhasha Andolon).

## Setup Instructions

To run this project locally:

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
